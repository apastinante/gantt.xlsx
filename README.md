# `gantt.xlsx`

Excel Gantt chart template with automatic:
- Gantt chart plot based on task dates
- Delay highlighting
- Task classification hierarchy
- Month labelling

> Tip: export Gantts to LaTeX with [Tables Generator](https://www.tablesgenerator.com)
